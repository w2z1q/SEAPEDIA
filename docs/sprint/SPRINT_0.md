# Sprint 0 - Project Foundation

## Objective

Membangun fondasi proyek SEAPEDIA agar seluruh tim memiliki environment, struktur project, dan database yang siap digunakan sebelum implementasi fitur.

---

# Scope

Sprint ini TIDAK membuat fitur aplikasi.

Sprint ini hanya berfokus pada setup project.

---

# Deliverables

- Backend Express.js berhasil dibuat.
- Frontend Next.js berhasil dibuat.
- Prisma berhasil terhubung ke PostgreSQL (Supabase).
- Swagger berhasil berjalan.
- Struktur folder sesuai standar.
- Git Repository siap digunakan.
- README awal dibuat.
- Environment variables dibuat.

---

# Acceptance Criteria

- Backend dapat dijalankan.
- Frontend dapat dijalankan.
- Prisma berhasil connect ke database.
- Tidak ada error saat menjalankan project.
- Folder project sesuai standar.
